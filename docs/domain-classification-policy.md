# Domain Classification Policy

Domain classes include social, directory, booking platform, marketplace, official candidate,
parked, suspicious, irrelevant, and unknown.

Initial social domains include Facebook, Instagram, LinkedIn, TikTok, X, and YouTube. Directory
examples include Yellow, Finda, Localist, Yelp, Tripadvisor, Foursquare, Cylex, and Hotfrog.
Lists are intentionally simple and should be updated after sample review.
