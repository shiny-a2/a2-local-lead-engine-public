# Phase 14 Persian Dashboard Policy

The Phase 14 dashboard is Persian, RTL, private/admin-only, and management-focused.

Pages:

- داشبورد پایلوت
- قیف کامل سیستم
- شاخص‌های کلیدی
- وضعیت فازها
- ریسک‌ها و هشدارها
- Fix Packها
- تصمیم توسعه / توقف / اصلاح
- آمادگی عملیاتی
- سیاست نگهداری داده
- گزارش نهایی پایلوت

The dashboard must not include send, follow-up, reply, lead collection, external API, price, quote, proposal, meeting, payment, or call actions.
