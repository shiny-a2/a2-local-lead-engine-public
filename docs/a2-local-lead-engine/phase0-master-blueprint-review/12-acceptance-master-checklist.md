# Acceptance Master Checklist

MVP readiness requires:

- [ ] Phase 1 foundation passes with fail-closed safety gates.
- [ ] Phase 2 source intake works within budgets and no Google Maps/Places.
- [ ] Phase 3 candidate normalization/dedupe quality is acceptable.
- [ ] Phase 4 verification and claim permissions are reliable.
- [ ] Phase 5 scoring/pilot selection gates low-risk candidates only.
- [ ] Phase 6 offer blocks are conservative and category-supported.
- [ ] Phase 7 drafts are evidence-bound and not sent.
- [ ] Phase 8 judge blocks unsafe/generic/misleading drafts.
- [ ] Phase 9 human approval dashboard is private and no-send.
- [ ] Phase 10 controlled sending enforces kill switch, unsubscribe, suppression, limits, locks, provider readiness.
- [ ] Phase 11 inbox/bounce reads only and creates tasks/statuses.
- [ ] Phase 12 opportunity planning is human-only.
- [ ] Phase 13 sales workspace is human-only.
- [ ] Phase 14 governance creates pilot QA and scale/no-scale decision.
- [ ] No unsafe automation exists.
- [ ] Dashboards are protected.
- [ ] No secrets exposed.
- [ ] Reports generated.
- [ ] Tests passing.
- [ ] MVP closure decision generated.
