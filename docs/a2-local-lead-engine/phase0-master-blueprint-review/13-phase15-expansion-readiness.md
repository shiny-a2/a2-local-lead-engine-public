# Phase 15 Expansion Readiness

Phase 15 must not be mixed into MVP.

Required expansion controls:
- country registry
- country compliance profile
- source strategy per country
- send policy per country
- localization profile
- dashboard language/RTL/LTR profile
- activation gates
- country statuses: DRAFT, REVIEW, ACTIVE, PAUSED, DISABLED
- per-country suppression and unsubscribe handling
- no automatic scale without country approval

Before any country activates:
1. Legal/compliance profile reviewed.
2. Source availability confirmed.
3. Send policy and email wording localized.
4. Dashboard labels reviewed.
5. Pilot volume approved.
6. Kill switch and country activation gate tested.

Phase 15 verdict should be separate from MVP readiness.
